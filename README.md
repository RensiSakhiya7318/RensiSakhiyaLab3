This Android application was developed as part of Lab 3.
The purpose of this project is to demonstrate the use of basic Android UI components,
intents, dialogs, image handling, and multi-language support.

## Features
- Displays a user interface using LinearLayout
- Allows alphabet-only input with a 10-character limit
- Opens device Settings using an implicit intent
- Opens a website using an implicit intent
- Rotates between four images using an ImageButton
- Displays a Toast message showing the image name
- Shows a Snackbar when the switch is turned ON or OFF
- Displays a non-dismissible AlertDialog when the back button is pressed
- Supports English and French languages
- Logs student name and ID when the app goes to background

## Technologies Used
- Android Studio
- Java
- Android SDK (API 31+)
- Material Design Components
